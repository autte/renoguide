# RenoGuide

A simple, conversion-focused landing page for homeowners in Canada to **avoid costly mistakes when replacing windows**.

## 🌐 Live Site

https://renoguide.net

---

## 🧠 What This Project Is

RenoGuide is a lightweight web page designed to:

* Help homeowners understand window replacement quotes
* Highlight hidden costs in installation
* Provide a structured checklist before signing a contract
* Educate users about rebates (e.g. Rénoclimat)
* Drive conversions to a paid checklist / guide

---

## 🎯 Target Users

* Homeowners in Montreal / Canada
* People currently comparing window quotes
* First-time renovators unsure about pricing and installation

---

## ⚙️ Tech Stack

* HTML (single-page)
* CSS (inline styles)
* Vanilla JavaScript
* Deployed via Vercel
* Domain managed via Namecheap

---

## 🚀 Deployment

This project is deployed using **Vercel with GitHub integration**.

### Steps:

1. Push code to GitHub
2. Import repo into Vercel
3. Vercel auto-deploys on every push
4. Custom domain (`renoguide.net`) is connected via DNS
